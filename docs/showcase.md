# Showcase

This page highlights a few example visual outputs from IntegraPose. These samples are meant to show the range of workflows the app can support, not to imply that every project follows the same assay design.

## Quick Tour

| Showcase area | What it demonstrates |
| --- | --- |
| Real-time overlays | Live or near-live visual feedback during tracking workflows |
| Rodent assay examples | Practical research-style behavior analysis outputs |
| Gait dashboard views | Downstream movement and kinematic summaries |
| Flexible video contexts | The same pipeline pattern applied across different recording styles |
